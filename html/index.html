Welcome to FastAPI Cluster Application

API Endpoints

This application provides a secure REST API for user management with high availability through clustering.

Cluster Information

This application runs on a cluster of 3 FastAPI instances with Nginx load balancing.

Each request is automatically routed to one of the available instances for better performance and reliability.

Available Routes:

  • Authentication: /api/token - Get JWT token
  • Register: /api/register - Create new user
  • Users: /api/users - Get all users
  • Current User: /api/users/me - Get current user
